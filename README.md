# VA Combined Disability Calculator

A single-page calculator that estimates a veteran's combined VA disability
rating from individual condition ratings, following the official VA Combined
Ratings Table formula (38 CFR §4.25) and the bilateral factor for paired
arms/legs (38 CFR §4.26).

## Usage
Open index.html in a browser. Add each rated condition, its percentage, and
(if applicable) which arm/leg it affects. The bilateral factor is applied
automatically when both sides of a pair are entered. The final combined
rating, rounded to the nearest 10%, is shown along with a step-by-step
breakdown of how it was calculated.

This is an estimation tool for informational purposes only and is not an
official VA determination.
